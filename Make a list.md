# Step 2: Make a list
## Markdown supports 3 types of common lists. They include:

### Unordered - Bulleted list
* Item 1
* Item 2
* Item 3
### Ordered - Number list
1. Item 1
2. Item 2
3. Item 3
### Task - Checkbox list
- [x] This task is complete
- [ ] This task is not complete
